# AI Starters for Rotary Leaders

A single-page web resource sharing ready-to-use AI prompt "starters" for club
leaders and officers — organized by challenge (running your club, growing
membership, events & fundraising, communications, and going further).

The page is a self-contained `index.html` file: open it in any browser, email
it, or host it anywhere. No build step or dependencies (the Manrope font loads
from Google Fonts).

## Usage

Before sharing, swap the two placeholder links in `index.html`:

- `REPLACE_WITH_SIGNUP_LINK` — your newsletter / updates form
- `REPLACE_WITH_FEEDBACK_LINK` — your feedback form
